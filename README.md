# TEMPERATURE CONVERTOR

A simple and user-friendly application to convert temperatures between **Celsius, Fahrenheit, and Kelvin**.  
This project started as a Python program and was transformed into a clean web-based GUI for better accessibility.

---

## Technology Used
* **Python** ‚Üí For Basic prototype 
* **HTML5** ‚Üí Structure of the web page  
* **CSS3** ‚Üí Styling and responsive design  
* **JavaScript (ES6)** ‚Üí Conversion logic and interactivity  

---

## Features
* Convert between **Celsius, Fahrenheit, and Kelvin** instantly  
* **Clean and simple GUI** with dropdown selection  
* **Validation** for invalid inputs (e.g., non-numeric values, negative Kelvin)  
* **Responsive design** that works on desktop and mobile  
* Clear and formatted results with two decimal precision  

---

## Process
* **Initial Program**: Wrote the temperature converter in Python (console-based).  
* **Web Transformation**: Rebuilt the logic using JavaScript for browser execution.  
* **GUI Design**: Designed a minimal, card-style interface with HTML & CSS.  
* **Hosting**: Deployed the project using GitHub Pages / Netlify for easy sharing.
